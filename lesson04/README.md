## js文件打包后随机生成hash值，解决文件缓存
> filename: '[name]_[hash:7].js'
> // name--> 指向entry对象的key名称 
> // hash--> 随机生成7位哈希值

### Windows中创建文件 type nul
`例如：type nul>home.js`

### Linux中创建文件 touch
`例如:touch home.js`

### Linux中查看当前工作目录
`pwd`
 
### Linux中新建文件夹 mkdir
`例如： mkdir src`

### Linux中查看文件内容 cat
`cat home.js` 

### 单独运行js文件
`cd src`
`node home.js`
